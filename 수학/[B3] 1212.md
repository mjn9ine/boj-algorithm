# [BOJ] 1212번 8진수 2진수

백준온라인저지(BOJ) :  https://www.acmicpc.net/problem/1212



## 1. 문제 설명

| 시간 제한 | 메모리 제한 | 
| :-------- | :---------- |
| 1 초      | 256 MB      | 

### 문제

8진수가 주어졌을 때, 2진수로 변환하는 프로그램을 작성하시오.

### 입력

첫째 줄에 주어진 수를 2진수로 변환하여 출력한다. 수가 0인 경우를 제외하고는 반드시 1로 시작해야 한다.

### 출력

가능한 D값의 최댓값을 출력한다.

#### 예제 입력 1

```
314
```

#### 예제 출력 1

```
11001100
```


## 2. 내 소스 코드

```python
import sys

input = sys.stdin.readline

if __name__ == "__main__":
    oct_num = int(input(), 8)
    print(bin(oct_num)[2:])
```
